# M5 Linux CLI

## Objective
Deliver a Linux-native Rust CLI for local session control and API/daemon integration.

## Acceptance Criteria
- [x] `crabbot codex` command supports start/resume/interrupt/status workflows.
- [ ] CLI can attach to local daemon and mirror session state updates.
- [x] CLI auth and config management work for local and remote API endpoints.
- [ ] End-to-end tests cover CLI-to-daemon and CLI-to-API happy path.
