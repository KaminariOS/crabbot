# M0 Foundation

## Objective
Establish a Rust monorepo baseline, workspace standards, and CI-ready guardrails.

## Acceptance Criteria
- [ ] Workspace layout matches planned `apps/`, `services/`, `crates/`, `schemas/`, and `docs/adr` structure.
- [ ] `cargo check --workspace`, `cargo test --workspace`, and lint entrypoints exist and run in CI.
- [ ] Core crate/service skeletons compile in development shell.
- [ ] ADR template and architecture decision process are documented.
